นโยบายความเป็นส่วนตัว สำหรับ RYW Latest

นโยบายความเป็นส่วนตัวฉบับนี้ (“นโยบาย”) อธิบายถึงวิธีการที่ RYW Latest เก็บรวบรวม ใช้ และเปิดเผย ข้อมูลส่วนบุคคล (“ข้อมูลส่วนบุคคล”) ที่ท่าน (“ผู้ใช้” หรือ “ท่าน”) ได้ให้สำหรับการใช้งานแอพพลิเคชั่น RYW Latest และผลิตภัณฑ์หรือบริการดังกล่าว (เรียกรวมกันว่า “เว็บไซต์” หรือ “บริการ”) เพื่อให้สอดคล้องกับพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (“PDPA”)

ซึ่งรวมไปถึงการอธิบายในตัวเลือกสำหรับท่านเกี่ยวข้องกับการใช้ข้อมูลส่วนบุคคลของท่านและวิธีการให้ท่านเข้าถึงข้อมูลหรือทำให้ข้อมูลเป็นปัจจุบัน นโยบายฉบับนี้ไม่บังคับใช้กับการปฏิบัติงานของบริษัทที่เราไม่ได้เป็นผู้ดูแลหรือบุคคลที่เราไม่ได้มีการจ้างหรือต้องจัดการ

แอพพลิเคชั่น RYW Latest เป็นแอพพลิเคชั่นสำหรับติดต่อกับเว็บไซต์โรงเรียนระยองวิทยาคม (rayongwit.ac.th)

แอพพลิเคชั่นนี้ทำงานได้อย่างไร?

ติดต่อกับเว็บโดยตรง (Android)

แอพพลิเคชั่น RYW Latest สำหรับระบบ Android จะทำการติดต่อกับเว็บไซต์โรงเรียนระยองวิทยาคมโดยตรง ข้อมูลที่ได้มาจะถูกแยกด้วยวิธี Web-Scrapping บนตัวแอพพลิเคชั่น

ติดต่อผ่านระบบ Proxy (PWA/iOS)

เนื่องจากข้อจำกัดในระบบเว็บ แอพพลิเคชั่น RYW Latest สำหรับ PWA (Progressive Web App) จะทำการติดต่อกับ RYWProxy เพื่อขอข้อมูลในเว็บไซต์โรงเรียนระยองวิทยาคม ข้อมูลที่ได้มาจะถูกแยกด้วยวิธี Web-Scrapping บนตัวแอพพลิเคชั่นเอง

RYWProxy เป็นระบบ Proxy สำหรับผู้ใช้ RYW Latest บน PWA/iOS ซึ่งไม่มีการเก็บข้อมูลใดๆ ทั้งสิ้น

ข้อมูลเพิ่มเติมที่อาจมีการเก็บ

การรายงานข้อบกพร่องของแอพพลิเคชั่น

แอพพลิเคชั่นจะมีการรายงานข้อบกพร่องผ่าน Google Play Console ได้แก่: โปรแกรมขัดข้องกะทันหัน (Crash) เป็นต้น เพื่อใช้ในการปรับปรุงแอพพลิเคชั่น ซึ่งข้อมูลที่พวกเราจะได้ จะไม่เจาะจงชื่อ/ตัวตนของผู้ใช้
